**A:** Run: `./run-qa.sh quick`

## 🎉 Quick Start (TL;DR)

```bash
# 1. Install
curl -fsSL https://mediar.ai/install.sh | bash
terminator setup

# 2. Setup
cd "/Users/donnysmith/Desktop/Tiger CMS/qa-automation"
pip3 install -r requirements.txt

# 3. Verify
python3 test_setup.py

# 4. Test
./run-qa.sh quick

# 5. View
open screenshots/*/
```
